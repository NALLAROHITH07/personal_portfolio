STUDENT COUNSELLING MANAGEMENT SYSTEM

Abstract :

STUDENT COUNSELLING MANAGEMENT is to automate and ease the burden and complexity of manual student counselling procedures. This may include tracking student details, student attendance details, academic results and additional activities such as certifications, internships etc. The manual system faces the challenge in proper data analysis and timely information identification. The proposed system will be web application which automates all these processes and ensures accurate and timely information.

Discription :

It is an real time Project which is useful to the college management and it is presently using by the BITS college management

Role in the project : Team Leader
Technology : Web based project
Technologies used : HTML5, CSS3, JS, PHP, Bootstrap
